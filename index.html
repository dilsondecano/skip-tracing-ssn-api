Enhanced People Search

The Enhanced People Search API helps you verify SSN owner and find contact information and background details for people in US.

Results typically include: names, aliases, current and previous addresses, phone numbers, email addresses, date of birth, date of death if deceased, relative names and DOB, SSN, SSN issue date and state. Driver license info is also available is some states.

Supported search input options:

  • SSN
  • Last 4 of SSN + First and Last Name (all 3 are required)
  • First and Last Name (middle name is optional) + DOB
  • First and Last Name (middle name and DOB are optional) + Address, City, State (zip is optional)
  • First and Last Name (middle name and DOB are optional) + City, State (zip is optional)
  • First and Last Name (middle name and DOB are optional) + State (zip is optional)
  • Address, City, State (zip is optional)
  • Phone Number

The API also supports search by last 4 of SSN, if both First and Last Name are provided.

This API is restricted and available only to verified businesses. Review Restricted Access application process below.

Cost is $0.77 to $0.30 per hit, depending on volume. No charge, if info not found.

The rate automatically goes down as your monthly volume increases. The price resets on the 1st of each month.

People Trace API

This API is a professional Skip Trace tool typically used by collection agencies to locate debtors and is now available to ALL our Verified Business Clients with Restricted Access. Results from this API typically include Names, Aliases (aka), Death Records Search, Current Address, Previous Addresses, and phone numbers associated with the SSN when available.

What makes this Skip Trace API the Skip Trace?

Online Skip Tracing from other vendors often turns up data from digital databases, but are you getting the right data? Results often depend on the data quality and the search technology. Searchbug has been online since 1998. We know data. Our Skip Trace services are used by government offices, law enforcement, private investigators, and many other analysts that require a technology that sorts, filters, matches, and looks for the high probability when delivering results.

This API is restricted and available only to verified businesses. Review Restricted Access application process below.

Cost is $1.97 per query.

Match Name and SSN API

This API is specifically designed to verify if an SSN provided by your customers, employees, or contractors matches their name. This tool is ideally suited for identity verification, issuing 1099s during tax season, and KYC compliance.

There are two options for verifying the Social Security Number (SSN): 1) submitting the last name and full 9-digit SSN or 2) submitting the first and last name and partial SSN (last 4 digits only).

If a match is found, the API returns the name associated with the SSN, recorded dates of birth, cities lived, state and year of issuance, and SSN status (active or deceased).

This API is available only to users approved for SSN Verification Access

Cost is $0.77 to $0.30 per query (match or no match), depending on volume.

The rate automatically goes down as your monthly volume increases. The price resets on the 1st of each month.

How do I get access to APIs?

  • To use an API you will need a Searchbug account.
  • The cost of each API is based on either a Query or a Hit (match) and the cost is deducted from the established Prepaid account balance. Review each API for the associated cost, and, if the charge is per Hit (when information is found) or per Query.
  • APIs require a Prepaid account. Prepaid accounts are automatically recharged when your account is low on funds to avoid service interruption. You can select any prepaid amount from $10 to $1,000. If you are just testing the system, we suggest starting with a $10 prepaid account. Your credit card will only be charged $10 and you can implement testing. When you finish testing, we suggest you increase your prepaid amount to match your anticipated volume to avoid multiple small charges.
  • Why it Is important to have the proper prepaid account amount selected: If you keep your prepaid account at $10 and anticipate using more than $10 in service every few days, your card will be charged multiple times in a short period. Multiple small charges to the same vendor in a short period can often trigger your credit card company to decline charges as suspicious activity which will cause your service to be interrupted.
  • We suggest you select a prepaid amount that will cover a least one week of search volume. Note that Prepaid amounts never expire. Should you decide to stop using the service, you can simply request the account balance of unused funds be credited back to your card.
  • If you need a restricted API, establish a Searchbug account first, then apply for Restricted Access.
